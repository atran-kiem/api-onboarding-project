# api-onboarding-project
Simple api project to learn the FeatherJS API


## Objectives
1. README with project setup instructions
2. Build minimal FeatherJS API in TypeScript
    * implement CRUD for a simple resource (e.g. todo items with title and completed fields)
3. Tests
    * Unit and integration tests to cover CRUD operations
    * 80% coverage
4. Github Actions
    * Add workflow that
        - runs on pushes and pull requests
        - installs dependencies, lints, and runs the tests
        - generates a coverage report
5. Coverage reporting
    * Integrate a free coverage service (e.g Codecov)
